# Portfolio Website

## Overview

This is a responsive personal portfolio website developed using HTML5 and CSS3. The website showcases personal information, skills, projects, and contact details in a clean and professional layout.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern navigation bar
* Home page with introduction section
* About page with education and skills information
* Contact page with contact details
* Project showcase section
* Hover effects and interactive UI elements
* Professional dark theme design

## Technologies Used

* HTML5
* CSS3
* GitHub Pages

## Pages

### Home Page

Contains a welcome section, introduction, project highlights, and skills overview.

### About Page

Provides information about education, interests, skills, and career objectives.

### Contact Page
